# Memoirix

## 📌 Description
Memoirix is a web-based personal diary application designed to help users digitally record their daily 
experiences, thoughts, and emotions in a structured and organized manner. The system aims to replace traditional 
handwritten diaries with a more efficient and accessible digital platform.

## 🚀 Features
- User login system
- Add daily mood entries
- View past records
- Simple and user-friendly interface

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
- Python
- Flask
- PostgreSQL
- Chart.js
